# 💈 BarberHub

Sistema web de agendamento para barbearias.

---

## ✨ Funcionalidades

- 🔐 Autenticação com dois perfis: **Administrador** e **Barbeiro**
- 👥 CRUD completo de clientes, barbeiros e serviços
- 📅 Agendamentos com controle de status (Agendado / Concluído / Cancelado) e validação de conflitos de horário
- 📊 Dashboard personalizado por perfil com estatísticas em tempo real
- 📄 Relatórios exportáveis em PDF

## 🚀 Como Executar

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/barberhub.git
cd barberhub

# 2. Instale e configure tudo automaticamente
composer run setup

# 3. Inicie o ambiente de desenvolvimento
composer run dev
```

Acesse: **http://localhost:8000**

> **Usuário padrão:** `admin@barberhub.com` / `password`  
> ⚠️ Altere as credenciais em produção!

---

## ⚙️ Requisitos

- PHP 8.3+, Composer
- Node.js 18+ e npm 9+
- MySQL 5.7+ ou SQLite
